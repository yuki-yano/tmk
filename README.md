# tmk
`tmk` easily kills tmux sessions using fzf

## Usage

### Command

```
$ tmk
```

### Widget

Allocate to shortcut key

```.zshrc
bindkey '^[k' tmk
```

## Installastion

#### antigen users

```
antigen bundle yuki-ycino/tmk
```

#### zgen users

```
zgen load yuki-ycino/tmk
```
